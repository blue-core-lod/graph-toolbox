Bulk Updates

To perform bulk updates, follow these steps:

  1. Load the graphs for those resources you want to update (limited by the available memory in your web-browser)
  2. Construct an Update SPARQL query to apply to the resources.
  3. Run Query and confirm that the changes reflect what you want updated.
  4. Save the resulting changed graph(s) to Blue Core
